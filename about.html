Fundamental and Application of Web Development

The course will first teach you HTML, CSS and Javascript as the fundamental languages to get you familiar with how webpage actually works. After the you grasp these pretty well by build a personal portfolio with your coding patners, you will learn libraies and frameworks, such as SASS, bootstrap, and React to help you build a website in a even faster way. In addtion, you will learn back end framworks Ruby on Rails to build a website with database which can store and manage user and activity data. After you can handle both front end and back end project, you are ready to apply these knowledge to build web application that solve real-world problem in all kinds of fields. The career center is there to help you connect with companies that has plenty of in-demands request from their customer. You job and opportunity are already there!
